# ⌨ CodeType — Typing Speed Test for Coders

A midnight-neon themed typing speed test built specifically for programmers. Practice real code snippets, track your WPM, see your mistake heatmap, and climb the leaderboard.

## Features

- **3 Modes** — Code snippets (JS, Python, C++, SQL, CSS), English words, Famous quotes
- **Live stats** — WPM, Accuracy, Timer, Errors, Streak
- **WPM Chart** — Real-time canvas graph of speed over time
- **Mistake Heatmap** — Visual keyboard showing your weakest keys
- **Leaderboard** — Top 10 scores saved locally
- **Tips page** — Home row guide, speed tips, coding-specific advice, WPM scale
- **Midnight Neon theme** — Cyan + pink neon on deep dark background
- **Tab to restart** — Instant restart with keyboard shortcut

## Deploy

1. Create repo `typing-speed-test` on GitHub
2. Upload: `index.html`, `style.css`, `typing.js`, `content.js`
3. Settings → Pages → main branch → Save
4. Live at `https://YOUR_USERNAME.github.io/typing-speed-test`

## Stack

Pure HTML + CSS + Vanilla JavaScript + Canvas API. No frameworks, no dependencies.
